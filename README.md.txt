# Dashboard interactif

Ce dashboard présente les indicateurs clés à partir des données d’enquête.

## Déploiement
Le site est accessible via GitHub Pages.

## Mise à jour des données
Pour actualiser :
1. Générer un nouveau fichier index.html
2. Remplacer l'ancien fichier dans le repository
3. Faire un commit + push

## Structure
- index.html : dashboard principal
- assets/ : images (logo, icônes)
